Big Design

https://github.com/sneubig/Big-Design

Summary: I wanted to create a project that focused on refining my JavaScript, while using my passion for CSS to create a realistic experience. I'm proud of the fact that no images were used for this project, it's all CSS styling.

Favorite Part: Making the cursor move around the laptop when you mouseover the laptops mousepad.

Hardest Part: Making the pencil look as if it was actually writing. I am really into using SVG images and animations, so I wanted to push myself to expand on my knowledge of SVG animations using CSS. I definitely underestimated the difficulty of this. I was hoping to not use any plug-ins, but given the time constraint, I ended up using pathAnimator.

Features to Add: When the user submits a search query, instead of opening a separate tab with the search results, I want the search results to show up within the laptops screen. After doing some research, I believe I'll have to use the Google Search API to retrieve the JSON data of the specific search criteria. Then I will have to style the data with CSS to look like the google search results page. Really excited and motivated to make this a reality!
